Computer Programmer



Overview

A computer programmer, also known as coders, codes the games so the computer can follow the instructions which involves more of a logical understanding more than an artistic one. Coders make the game work the way it was intended to. They allow control to the player so they can do the thngs that most games have like the ability to move in different direction, jump up, look around, and interact with the enviroment around them. However, they do much more than just worry about the commands which is why they have multiple people that are segmented to do different tasks. They make the psychics of the game. This includes things like how fast the player falls or how a car accelerations, or even how fast a bullet travels


Although it takes multiple roles to make a game. A company that creates games that don't have a computer programer is a company that won't have an game. It's one thing to create the look of the game and make it look phenomenal but if there's no code so any one can play the game. It will basically be a screensaver because there needs to be code to enable animations of any sort or for a menu to work. No code equals no game.

Education Needed

Many game development companies prefer to hire programmers who have at least a bachelor's degree. Several degree programs might prepare you for the field, including those more general programs in computer programming and mathematics. However some schools do offer specialized degree programs, such as a Bachelor of Science in Game Programming. A 4-year degree program in game programming teaches you how to design, program and test new software. Such programs often require you to create your own video game program before graduation. Some specific courses covered include visual design, programming, game development, computer graphics and game modification. A bachelor's degree program in software engineering will also provide you with the basic building blocks necessary to work in the video game programming field.

Tips

  • 1. Get friendly with commercially available game engines
  • 2. Learn different programming languages
  • 3. Practice as much as you can
  • 4. Make your own programs to show to employers
  • 5. Get use to being in front of a screen a lot
